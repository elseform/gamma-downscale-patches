# 2K - GAMMA Downscale Pack [Main] V2506

Texture downscale pack for lower-RAM S.T.A.L.K.E.R. Anomaly GAMMA installs.

This pack caps targets large weapon / devices / misc textures at `2048px` on the longest edge.

## Install

Install as a normal MO2 mod. It must load lower than the source mods it overwrites, so the downscaled DDS files win conflicts.

## Scope

The pack covers selected large texture winners from the current GAMMA install, including weapons, actor hands, some items, some optic/attachment textures, and miscellaneous non-UI textures.
