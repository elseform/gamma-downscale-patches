# 1K - GAMMA Downscale Pack [Weapon Attachments] V2506

Texture downscale pack for lower-RAM S.T.A.L.K.E.R. Anomaly GAMMA installs.

This pack targets weapon attachment/module textures and downscales them to `1024px` on the longest edge.

## Install

Install as a normal MO2 mod. It must load lower than the source mods it overwrites, so the downscaled DDS files win conflicts.

## Scope

The pack covers scope/optic bodies, suppressors, muzzle attachments, tactical devices, grenade launcher attachments, and small weapon accessories.
